# Notes

这是我的个人长期笔记库，用来保存学习笔记、项目拆解、外部资料和阶段复盘。

## 目录结构

- `index/`：人看的索引入口。
- `topics/`：通用知识笔记。
- `projects/`：长期项目、产品和系统设计。
- `reference/`：外部资料原文。
- `inbox/`：临时输入和未归类想法。
- `reviews/`：阶段复盘。
- `archive/`：旧内容归档。

## 整理原则

- 保留原意，不要擅自改写成标准教材口吻。
- 不要为了分类而分类。
- 外部资料放 `reference/`。
- 项目推进放 `projects/`。
- 通用知识沉淀放 `topics/`。
- 临时想法先放 `inbox/`。
