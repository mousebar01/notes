# FastAPI 开发专题

此目录包含关于 FastAPI 框架的核心概念、基础使用和进阶技巧的笔记。

- [FastAPI基础教程](./FastAPI基础教程.md)：更详尽的基础知识，包含类型提示、路径/查询参数、请求体处理、响应配置和错误处理。
- [FastAPI依赖注入](./FastAPI依赖注入.md)：深入探讨 FastAPI 的依赖注入（Dependency Injection）机制，包含 `Depends` 使用、类作为依赖、子依赖、依赖覆盖测试以及在路由与全局的应用。
