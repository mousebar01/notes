# 手撕注意力机制

这篇笔记记录一版最小可运行的 causal self-attention。重点不是把 Transformer 全部写完，而是把注意力里最容易写错的几个地方弄清楚：

- `QK^T` 的 shape
- 为什么要除以 `sqrt(d)`
- causal mask 怎么写
- softmax 应该在哪个维度做
- attention weights 最后应该乘 `V`
- 多头注意力如何拆头和合并

## 1. 单头 scaled dot-product attention

最核心的注意力公式是：

```text
Attention(Q, K, V) = softmax(QK^T / sqrt(d_k)) V
```

对应代码：

```python
import torch
import torch.nn.functional as F


def attention(q, k, v, mask=None):
    """
    q, k, v: (..., T, D)
    mask:   (..., T, T)，True 表示可见，False 表示屏蔽
    """
    d = q.size(-1)

    scores = q @ k.transpose(-2, -1)
    scores = scores / (d ** 0.5)

    if mask is not None:
        scores = scores.masked_fill(mask == 0, float("-inf"))

    weights = F.softmax(scores, dim=-1)
    out = weights @ v

    return out
```

这里要注意几个点：

```text
q:       (..., T, D)
k^T:     (..., D, T)
scores:  (..., T, T)
weights: (..., T, T)
v:       (..., T, D)
out:     (..., T, D)
```

softmax 要在最后一维做，也就是对每个 query 位置，看它应该关注哪些 key 位置。

## 2. Causal attention

GPT 这类 decoder-only 模型不能看到未来 token，所以需要 causal mask。

假设序列长度是 `T=4`，每个位置可见范围是：

```text
第 0 个 token：只能看 0
第 1 个 token：能看 0, 1
第 2 个 token：能看 0, 1, 2
第 3 个 token：能看 0, 1, 2, 3
```

对应下三角 mask：

```text
1 0 0 0
1 1 0 0
1 1 1 0
1 1 1 1
```

代码：

```python
def causal_attention(q, k, v):
    """
    q, k, v: (B, H, T, D)
    """
    B, H, T, D = q.shape

    scores = q @ k.transpose(-2, -1)
    scores = scores / (D ** 0.5)

    mask = torch.tril(
        torch.ones(T, T, device=q.device, dtype=torch.bool)
    )

    scores = scores.masked_fill(~mask, float("-inf"))
    weights = F.softmax(scores, dim=-1)
    out = weights @ v

    return out
```

这里 `mask` 的 shape 是 `(T, T)`，会自动 broadcast 到 `(B, H, T, T)`。

## 3. 多头自注意力

输入 `x` 的 shape 是：

```text
x: (B, T, C)
```

其中：

```text
B = batch size
T = sequence length
C = n_embd
```

多头注意力会把 `C` 拆成：

```text
C = n_head * head_dim
```

例如：

```text
n_embd = 768
n_head = 12
head_dim = 64
```

完整代码：

```python
import torch
import torch.nn as nn
import torch.nn.functional as F


class MultiHeadSelfAttention(nn.Module):
    def __init__(self, n_embd, n_head):
        super().__init__()
        assert n_embd % n_head == 0

        self.n_head = n_head
        self.head_dim = n_embd // n_head

        self.q_proj = nn.Linear(n_embd, n_embd)
        self.k_proj = nn.Linear(n_embd, n_embd)
        self.v_proj = nn.Linear(n_embd, n_embd)
        self.out_proj = nn.Linear(n_embd, n_embd)

    def forward(self, x):
        """
        x: (B, T, C)
        """
        B, T, C = x.shape

        q = self.q_proj(x)
        k = self.k_proj(x)
        v = self.v_proj(x)

        q = q.view(B, T, self.n_head, self.head_dim).transpose(1, 2)
        k = k.view(B, T, self.n_head, self.head_dim).transpose(1, 2)
        v = v.view(B, T, self.n_head, self.head_dim).transpose(1, 2)

        # q, k, v: (B, H, T, D)
        scores = q @ k.transpose(-2, -1)
        scores = scores / (self.head_dim ** 0.5)

        mask = torch.tril(
            torch.ones(T, T, device=x.device, dtype=torch.bool)
        )
        scores = scores.masked_fill(~mask, float("-inf"))

        weights = F.softmax(scores, dim=-1)
        out = weights @ v

        # out: (B, H, T, D) -> (B, T, H, D) -> (B, T, C)
        out = out.transpose(1, 2).contiguous().view(B, T, C)
        out = self.out_proj(out)

        return out
```

## 4. Transformer Block

注意力层通常会和 MLP、LayerNorm、残差连接组合成一个 Transformer Block。

这里用的是常见的 pre-norm 写法：

```python
class MLP(nn.Module):
    def __init__(self, n_embd):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(n_embd, 4 * n_embd),
            nn.GELU(),
            nn.Linear(4 * n_embd, n_embd),
        )

    def forward(self, x):
        return self.net(x)


class TransformerBlock(nn.Module):
    def __init__(self, n_embd, n_head):
        super().__init__()
        self.ln1 = nn.LayerNorm(n_embd)
        self.attn = MultiHeadSelfAttention(n_embd, n_head)
        self.ln2 = nn.LayerNorm(n_embd)
        self.mlp = MLP(n_embd)

    def forward(self, x):
        x = x + self.attn(self.ln1(x))
        x = x + self.mlp(self.ln2(x))
        return x
```

结构可以理解成：

```text
x
-> LayerNorm
-> Multi-Head Self-Attention
-> residual add
-> LayerNorm
-> MLP
-> residual add
```

## 总结

注意力机制最关键的计算链路是：

```text
x
-> q, k, v
-> scores = q @ k^T / sqrt(head_dim)
-> causal mask
-> weights = softmax(scores)
-> out = weights @ v
-> 合并多头
-> out projection
```

如果手写时只记一个核心点，就是：

> `QK^T` 只是算注意力分数，softmax 后得到权重，真正被加权汇总的是 `V`。
